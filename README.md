# Projeto-Criador-de-Sites
Gerador de Páginas com IA: aplicação web onde o usuário digita a descrição de um negócio e a inteligência artificial (Llama 3.3 via API do Groq) gera o HTML e CSS de uma landing page personalizada. A interface exibe simultaneamente o código-fonte gerado e uma pré-visualização interativa do site em tempo real através de um iframe.
